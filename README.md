## 📂 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Gayan-Withanage/Snake_game-python.git
```

2. Navigate to the project folder:

```bash
cd Snake_game-python
```

3. Install dependencies:

```bash
pip install pygame
```

4. Run the game:

```bash
python main.py
```
